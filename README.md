## 🚀 编译与运行

### 单个文件编译
```bash
gcc -o output file.c        # C 文件
g++ -o output file.cpp      # C++ 文件
./output                    # 运行
```

### 批量编译（所有 .c 文件）
```bash
for f in */*.c; do gcc -o "${f%.c}" "$f" 2>/dev/null; done
```

### 使用 Makefile（推荐）
```bash
make all    # 编译所有
make clean  # 清理
```

## 📚 学习建议

1. **入门阶段**：从 `数学_逻辑/` 和 `数组/` 开始，掌握基本语法
2. **进阶阶段**：学习 `字符串/` 和 `排序/`，理解常用算法
3. **深入阶段**：研究 `数据结构/` 和 `c 语言学习/`，提升工程能力
4. **挑战阶段**：尝试 `递归/` 和 `位运算/` 中的高阶题目
