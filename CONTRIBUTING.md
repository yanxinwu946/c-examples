# 贡献指南

欢迎贡献！以下是提交新题目的规范。

## 提交新题目

1. **分类放置**：将文件放入对应的分类目录（如 `数学_逻辑/`、`字符串/` 等）
2. **命名规范**：使用有意义的中文或英文文件名，如 `素数判断.c`
3. **代码规范**：
   - 使用标准的 C99/C11 语法
   - 添加必要的注释说明题目和解题思路
   - 确保代码可编译运行
4. **提交格式**：
   ```
   feat: 添加 xxx 题目

   - 题目描述：xxx
   - 所属分类：xxx
   ```
